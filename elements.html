Port Scanner - Project Overview

This is a Python-based multi-threaded port scanner that can scan both TCP and UDP ports on target IP addresses. The project is designed to efficiently identify open or filtered ports using concurrent scanning.

Project Files

  • Download README.md
  • Download Source Code (port_scanner.py)
  • Download Instructions (port_scanner_instructions.txt)

Installation

No additional libraries are required, but you will need Python 3.2 or higher. You can check your Python version with:

python --version

Usage

  1. Clone the repository to your local machine.
  2. Run the script using Python:
    3. python port_scanner.py
  3. Input the target IP address and port range when prompted.

Functionality

  • Scans both TCP and UDP ports on the specified target IP.
  • Uses multi-threading to scan multiple ports concurrently for faster results.
  • Outputs the status of each port (open or filtered).

Future Improvements

Potential improvements include adding ICMP error handling, service detection, and banner grabbing functionality for more in-depth scanning.
